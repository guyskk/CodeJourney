###目录结构

除了 <code>node_modules</code> 中的文件，所以有的源代码都放在 Github 上。其目录结构：

    musicPlayers/  
        |-- app                  存放html文件  
        |-- static               存放静态文件  
        |   |-- asssets          less  
        |   |-- src              javaScript  
        |   |-- build            发布使用的js，css 文件  
        |   |   |-- css  
        |   |   |-- scripts  
        |-- files                暂时存放音乐文件
